## Page Sections

Individual sections you can mix and match to build any page.

### Heroes

The first thing visitors see. Sets the tone for the entire page.

| # | Hero Type | Description | Best For |
|---|-----------|-------------|----------|
| 1 | **Headline + CTA** | Large headline, subtitle, single prominent CTA button. Clean and direct. | SaaS, startups, any product with a clear value prop |
| 2 | **Headline + Image/Mockup** | Split layout — headline left, product screenshot or mockup right | SaaS, apps, tools — show the product immediately |
| 3 | **Video Background** | Full-width background video (muted, auto-play) with text overlay | Agencies, luxury brands, immersive experiences |
| 4 | **Animated/Motion Hero** | Text or elements animate in on load — typing effects, parallax, reveals | Creative agencies, portfolios, tech products |
| 5 | **Full-Bleed Image** | Full-screen hero image with overlaid headline and CTA | Photography, travel, food, lifestyle brands |
| 6 | **Gradient/Abstract** | Bold gradient or abstract art background with centered text | AI products, crypto, modern SaaS, creative tools |
| 7 | **3D/Interactive** | WebGL, three.js, or interactive 3D element as the hero centerpiece | Tech products, gaming, high-end showcases |
| 8 | **Split Screen** | Two equal halves — image on one side, content on the other | Comparison products, dual-audience pages, portfolios |
| 9 | **Minimal Text Only** | Just a headline and maybe a subtitle. Maximum whitespace. | Luxury brands, minimalist products, personal sites |
| 10 | **Product Configurator** | Interactive hero where users can customize/configure a product | E-commerce, automotive, fashion, custom products |
| 11 | **Search-First** | Large search bar as the primary hero element | Marketplaces, directories, documentation, job boards |
| 12 | **Waitlist/Countdown** | Coming soon countdown timer with email capture form | Pre-launch products, events, limited releases |
| 13 | **Stats/Numbers** | Large animated counters or stats as the hero visual | Enterprise, B2B, trust-focused products |
| 14 | **Carousel/Slider** | Multiple hero slides rotating through different messages | E-commerce, multi-product companies, news |
| 15 | **Bento Hero** | Apple-style bento grid layout in the hero area showing multiple features | Tech products, feature-rich apps, SaaS |
| 16 | **AI/Dynamic** | Personalized hero that adapts based on user input or context | AI products, smart platforms, recommendation engines |
| 17 | **Scroll-Triggered** | Hero content reveals progressively as user scrolls | Storytelling brands, immersive narratives, luxury |
| 18 | **App Store Style** | Device mockup centered with app screenshots and download buttons | Mobile apps, app landing pages |

### Pricing

Layouts for presenting plans, tiers, and pricing information.

| # | Pricing Layout | Description | Best For |
|---|---------------|-------------|----------|
| 1 | **3-Tier Cards** | Classic three cards side by side (Free / Pro / Enterprise) with feature lists | SaaS, subscriptions, any tiered product |
| 2 | **2-Tier Simple** | Just two options — free vs paid, or monthly vs annual | Micro SaaS, simple products, indie tools |
| 3 | **Comparison Table** | Full feature comparison grid below or instead of cards | Complex products with many feature differences |
| 4 | **Toggle (Monthly/Annual)** | Price cards with an animated toggle switch for billing period | SaaS with annual discount incentive |
| 5 | **Slider/Calculator** | Interactive slider that calculates price based on usage | Usage-based pricing, API products, cloud services |
| 6 | **Single Price + Features** | One price prominently displayed with a feature list below | Single-tier products, tools, courses |
| 7 | **Enterprise Contact** | Cards for standard tiers + "Contact Sales" for enterprise | B2B SaaS with enterprise segment |
| 8 | **Highlighted/Recommended** | Middle tier visually elevated (larger, colored border, "Most Popular" badge) | Any tiered pricing — drives mid-tier conversion |
| 9 | **FAQ Under Pricing** | Pricing cards followed immediately by pricing-specific FAQ accordion | Products where pricing needs explanation |

### Features

Sections that showcase what your product does.

| # | Feature Layout | Description | Best For |
|---|---------------|-------------|----------|
| 1 | **Icon Grid (3x2 or 4x2)** | Grid of features, each with an icon, title, and short description | SaaS, apps — the classic feature showcase |
| 2 | **Alternating Image/Text** | Feature rows that alternate image-left/text-right, then image-right/text-left | Products that benefit from screenshots per feature |
| 3 | **Bento Grid** | Apple-style mixed-size cards highlighting features at different visual weights | Tech products, feature-rich platforms |
| 4 | **Single Feature Spotlight** | One feature takes up the entire section with large visual and detailed description | Flagship feature, key differentiator |
| 5 | **Tabs/Accordion** | Features organized into tabs or expandable sections by category | Products with many features across categories |
| 6 | **Scroll-Reveal** | Features animate in as the user scrolls down the page | Storytelling approach, engaging presentations |
| 7 | **Comparison (Before/After)** | Side-by-side or slider showing the problem vs your solution | Products that solve a visible problem |
| 8 | **Interactive Demo** | Live embedded demo or interactive walkthrough of features | Developer tools, SaaS, complex products |
| 9 | **Stats + Features** | Features paired with impressive numbers (99.9% uptime, 10x faster) | Enterprise, performance-focused products |
| 10 | **Timeline/Steps** | Features presented as a numbered process or journey | Products with a workflow or onboarding sequence |
| 11 | **Video Showcase** | Short video clips demonstrating each feature | Visual products, design tools, creative platforms |
| 12 | **Card Carousel** | Horizontally scrollable cards, each highlighting a feature | Mobile-first, many features in compact space |

### Accessibility Ranking

| Grade | Charts | Notes |
|-------|--------|-------|
| **AAA** | Bar, Bullet | Safe for any context |
| **AA** | Line, Funnel, Gauge, Waterfall, Radar, Anomaly, Box Plot, Waffle, Forecast, Decomposition | Good with proper labels |
| **B** | Scatter, Heatmap, Geographic, Streaming, Process, Candlestick | Need table alternatives |
| **C** | Pie/Donut, Treemap, Sankey, Sunburst, Word Cloud | Poor — always provide alternatives |
| **D** | Network Graph, 3D Spatial | Fundamentally inaccessible — must have alternative views |
